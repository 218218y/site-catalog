# תכנית השקה עתידית לחיפוש Google

## מצב המימוש

תשתית ההכנה כבר קיימת, אך האתר נשאר במכוון במצב `private`:

- כל דף מקבל `noindex` גם ב־HTML וגם בכותרת `X-Robots-Tag`.
- לא נוצר sitemap במצב הפרטי.
- נוצרו כתובות נקיות לעמודי קטגוריה, קטלוג ועמוד מדויק.
- לכל עמוד יש canonical, Open Graph, תמונת שיתוף ו־JSON-LD שנוצרים בזמן הבנייה.
- דף הבית מכיל קישורי קטגוריות וקטלוגים כבר ב־HTML הראשוני.
- מעבר ל־public מחייב אישור כפול ומפורש.

הוראות מפורטות נמצאות ב־[`seo-build-modes.md`](seo-build-modes.md).

## מה יקרה ביום ההשקה

1. להריץ בדיקות מלאות ובאנדל private אחרון.
2. לבנות public מקומית ולבדוק title, canonical, robots, structured data ו־sitemap.
3. לוודא שה־CDN מחזיר את תמונות השיתוף ללא חסימה.
4. לפרסם באמצעות מצב public המפורש.
5. לאמת בעלות ב־Google Search Console ולשלוח sitemap.
6. להתחיל במספר קטן של עמודי קטגוריה וקטלוג ולבדוק את תוצאות האינדוקס.

## עמודים שיישארו מחוץ לאינדקס גם לאחר ההשקה

- מועדפים ורשימות משותפות;
- כתובות viewer כלליות;
- כל עמוד מדויק מתוך קטלוג (`/catalog/.../page/.../`);
- מסכי עזר, ניהול וכלי פיתוח.

כך קישורי שיתוף יכולים להציג תמונת דגם וכותרת מדויקת, בלי להפוך מאות עמודי
תמונה לתוצאות חיפוש נפרדות.

## שערי release שהושלמו

- H1 יחיד בכל עמוד שמיועד לאינדוקס, בלי להוסיף hero חדש בדף הבית;
- נעילת מזהי קטלוגים ו־slugs בקובץ `seo-routes.lock.json`;
- public preview שנבנה ונבדק אוטומטית ב־CI;
- בדיקת קישורים פנימיים, canonical, Open Graph, JSON-LD ו־sitemap;
- כלי בדיקה חיצוני ל־metadata ולתמונות השיתוף לאחר deployment;
- artifact ציבורי זמני ב־GitHub Actions לצורך סקירה, ללא פרסום אוטומטי.

פרטי הפקודות ורשימת ההשקה נמצאים ב־`public-search-release.md`.
