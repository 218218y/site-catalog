נכון לעכשיו האתר מוגדר להיות מוסתר מחיפוש גוגל

התקנה מקומית ראשונה
setup-windows.bat

לוח שליטה נוח בדפדפן לכל פעולות הקטלוגים
catalog-control-panel.bat

סדר עבודה מומלץ למסלול שרת אתר + Cloudflare R2
1. אחרי הוספת PDF חדש לתיקיית assets\pdfs מריצים:
   sync-catalog-pdfs.bat

2. עורכים את פרטי הקטלוג בלוח השליטה או בקובץ:
   catalogs.config.json

3. ממירים PDFים חדשים/שהשתנו לתמונות ואינדקס חיפוש:
   convert-catalogs.bat

4. בודקים מראש מה ישתנה ב-R2 בלי לבצע שינוי אמיתי:
   sync-r2-images-preview.bat

5. מסנכרנים תמונות ל-R2 בפועל: מעלה חדשות/שהשתנו ומוחק מה-bucket מה שכבר לא קיים מקומית
   sync-r2-images.bat

6. יוצרים תיקיית העלאה לשרת אתר באנדל:
   bundle-site-r2.bat

7. מעלים ל-Cloudflare Pages את הבאנדל:
bundle-site-r2-upload cloudflare.bat

הפקודה שהקובץ מריץ בפועל:
npx --yes wrangler pages deploy "dist\site-upload-r2" --project-name bargig-catlog --branch main

אם רוצים ליצור באנדל חדש ומיד להעלות:
bundle-site-r2-upload cloudflare.bat --build-first


פקודות שימושיות נוספות

סנכרון תמונות ל-R2 בפועל: מעלה חדשות/שהשתנו ומוחק מה-bucket מה שכבר לא קיים מקומית
sync-r2-images.bat

סנכרון R2 בלי מחיקות
sync-r2-images.bat --no-delete


המרה של כל PDF מחדש בהכרח, בלי למחוק קטלוג שכבר נמחק PDF שלו
convert-catalogs-force.bat

המרת PDF חדשים בלבד + מחיקת קטלוגים שנמחקו מתוך catalogs.config.json
convert-catalogsdelete.bat

המרה של כל PDF מחדש בהכרח + מחיקת קטלוגים שנמחקו מתוך catalogs.config.json
convert-catalogs-deleteforce.bat

רענון מחדש רק של זיהוי OCR/חיפוש
refresh-ocr-search.bat

שרת מקומי לאתר הראשי בלבד
start-server.bat
או
npx serve .

לוח השליטה אינו עובד דרך שרת האתר הראשי. בשבילו מריצים:
catalog-control-panel.bat

מחיקה דרך לוח השליטה מסירה רק מהרשימה ומהאתר שנוצר, לא מוחקת PDF ולא מוחקת assets/pages.
לניקוי תיקיות תמונות ישנות מריצים:
convert-catalogsdelete.bat

הגדרת אתר ב-Cloudflare Pages
Site configuration / Domain management / Production domains

bargig-furniture.com
www.bargig-furniture.com

Verify DNS configuration

הגדרת OCR פר קטלוג או על ידי לוח השליטה או בקובץ catalogs.config.json אפשר להוסיף לכל קטלוג שדה:
"ocr": false
ואז גם אם מריצים המרה רגילה, הקטלוג הזה לא יריץ OCR. ברירת המחדל היא true. אין צורך בפקודת המרה גלובלית בלי OCR.

הערת R2 חשובה
קובץ r2.env נשאר מקומי בלבד ומכיל מפתחות גישה. לא מעלים אותו ל-Cloudflare Pages ולא ל-GitHub.


שינוי תקיות r2 קלאוד - מונע בדיקה חדשה של נטפרי
בדיקה
rclone copy "r2:bargig-catalog/assets/pages/bedrooms-gober-gallery" "r2:bargig-catalog/assets/pages/bedrooms-guber-gallery" --dry-run -P
העתקת תקייה
rclone copy "r2:bargig-catalog/assets/pages/bedrooms-gober-gallery" "r2:bargig-catalog/assets/pages/bedrooms-guber-gallery" -P
מחיקת תקייה ישנה
rclone delete "r2:bargig-catalog/assets/pages/bedrooms-gober-gallery" --dry-run -v
rclone delete "r2:bargig-catalog/assets/pages/bedrooms-gober-gallery" -P
